# Bee Creek Farm — Brand Spec

## Color palette

| Token            | Hex       | Use                                                                 |
|------------------|-----------|---------------------------------------------------------------------|
| Honey Gold       | `#E5A93A` | Primary brand color. Bee body, flower petals, primary buttons.      |
| Honey Light      | `#F0BE55` | Petal highlight, hover state.                                       |
| Mason Sky        | `#7AA8C7` | Hill Country sky, cool secondary accent.                            |
| Mesquite Green   | `#5B6E3A` | Foliage, footer, deep accent.                                       |
| Wildflower Red   | `#B5482E` | Indian blanket flower accent. Use sparingly for emphasis.           |
| Bone             | `#F4EFE3` | Page background, off-white text on dark.                            |
| Ink              | `#1F1B16` | Body copy, bee outline, dark sections.                              |

Pulled directly from the property photography — gold matches the silo flower painting, sky pulled from the hero photo, mesquite from foliage averages, wildflower red from the Indian blanket bloom in the foreground of the silo shot.

## Typography

- **Display:** "Fraunces" (Google Fonts) — warm, slightly rustic serif with optical sizing. Use for H1/H2 and the wordmark in HTML lockups.
- **Body:** "Inter" (Google Fonts) — humanist sans, neutral, highly legible. Use for paragraphs, UI, captions.

Type scale (rem):
- Display XL: 4.5 / Display L: 3.25 / Display M: 2.25 / H3: 1.5 / Body: 1.0625 / Small: 0.875 / Caption: 0.75

Tracking: tight on Fraunces display (`-0.02em`), neutral on Inter, wide on small caps labels (`0.16em`).

## Logo lockups

- `logo-primary.svg` — stacked icon + wordmark + Mason, TX. Default for hero, posters, large prints.
- `logo-horizontal.svg` — icon left + wordmark right. Default for site headers, email signatures.
- `logo-icon.svg` — bee-in-flower only. Social avatar, app icon, watermark.
- `logo-mono-black.svg` / `logo-mono-white.svg` — single-color variants for stamps, embroidery, dark photo overlays.
- `favicon.svg` — rounded dark-square version for browser tabs.

Minimum sizes: icon never below 32 px on screen, never below 0.5 in in print. Maintain padding equal to one petal width.

## Voice

Warm, plain-spoken, Texan but unforced. Short sentences. Specifics over slogans ("32 acres along Bee Creek" beats "Your perfect getaway"). Never use "luxury," "experience," "escape" as a noun.
